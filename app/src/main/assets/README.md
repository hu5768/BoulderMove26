# MediaPipe Model File

Please download the MediaPipe Pose Landmarker Lite model file:

1. Go to: https://developers.google.com/mediapipe/solutions/vision/pose_landmarker#models
2. Download `pose_landmarker_lite.task`
3. Place it in this directory: `app/src/main/assets/pose_landmarker_lite.task`

The model file is required for pose detection to work.
